annis.visualizers.component.pdf.PDFFullVisualizer Class Reference

Detailed Description

Renders pdf documents with pdf.js.

Fetches the whole pdf document, could be slow since the underlying pdf.js do not support lazy loading yet.

For large pdfs it might be convenient, to set inner scrollbars with via the mappings in the resolver_vis_tab file:@code height:<Integer>
