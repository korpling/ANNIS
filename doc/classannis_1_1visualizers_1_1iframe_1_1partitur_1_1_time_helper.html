Split a time annotation s.ms-(s.ms)? Whether the flag first is set to true, we return the first value, otherwise we did try to return the second.

The end time don't have to be annotated, in this case it returns "undefined". Without a defined start time the result is an empty String "undefined". If there is no time anno, it returns undefined
