annis.visualizers.component.graph.DotGraphVisualizer Class Reference

Detailed Description

A debug visualization of the annotation graph.

Mappings:
Use all_ns:true to visualize the entire annotation graph. Specifying e.g. node_ns:tiger or edge_ns:tiger instead causes only nodes and edges of the namespace tiger to be visualized (i.e. only a subgraph of all annotations).
