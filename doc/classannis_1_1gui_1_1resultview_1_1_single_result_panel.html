void annis.gui.resultview.SingleResultPanel.registerVisibilityStatus ( long  entryId,
boolean  status 
)

Registers the visibility status of single visualizer.

The ordinary state is determined by the resolver entry, which is usually provided by the ResolverProvider. If this status is changed, it must be propagated to an instance of the interface.

Parameters
entryIdThe entry id, which represents the row id in the resolver map table, thus this value should be unique.
statusIf true the visualizer is visible.
