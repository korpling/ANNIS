boolean annis.gui.widgets.grid.Row.merge ( Row  other) throws IllegalArgumentException

Merges the other row into this row.

This means all events from the other Row are added to this row. The other Row will not be changed. Only rows which have no overlapping events can be merged.

Parameters
otherThe other Row which will be merged into this one.
Returns
Returns true if merge was successfull, false if no merge was done.
